> **CodeCure Hackathon — Track A: Drug Toxicity Prediction**

A research-grade drug toxicity prediction system combining Graph Attention Networks, XGBoost, SHAP explainability, counterfactual molecule generation, and Claude AI reasoning.

## 🚀 Quick Start

### 1. Install dependencies
```bash
# Python 3.9+ recommended
pip install -r requirements.txt

# RDKit (if not installed via pip)
conda install -c conda-forge rdkit
```

### 2. Train models
```bash
python train.py
# Takes ~20 min on CPU, ~5 min on GPU
# Saves to models/gnn_model.pt and models/xgb_model.pkl
```

### 3. Set your API keys
Create a `.env` file in the root directory:
```env
# Add whichever keys you prefer to use
ANTHROPIC_API_KEY="sk-ant-..."
GEMINI_API_KEY="AIzaSy..."
```

### 4. Launch the app
```bash
streamlit run app.py
```
Open http://localhost:8501

## 🧠 What Makes This Different

### 1. Dual-Model Ensemble
- **GAT** captures ring topology, bond patterns, aromatic systems — structural features that matter for toxicity
- **XGBoost** on ECFP fingerprints captures global statistical patterns
- Weighted ensemble outperforms either model alone

### 2. Full Explainability Stack
- **SHAP values** on XGBoost: which molecular bits push predictions up/down
- **GAT attention weights**: which atoms the network "focuses on" when predicting toxicity
- Both are visualised atom-by-atom on the molecular structure

### 3. ★ Counterfactual Molecule Editor
Given a toxic molecule, the system:
1. Identifies toxic substructures (nitro groups, epoxides, aldehydes, etc.)
2. Applies RDKit-based structural modifications (remove –NO₂, open epoxide, reduce logP)
3. Re-predicts each candidate through the full ensemble
4. Ranks and displays safer alternatives with per-endpoint improvement breakdown

This is what separates a *model* from a *drug discovery tool*.

### 4. LLM / AI Layer (Claude & Gemini)
- Plain-English explanation of *why* a molecule is toxic (mechanism, biological targets)
- Per-counterfactual explanation of *why* a modification works
- Full structured research report (downloadable Markdown)
- Multi-turn conversational Q&A about the molecule

## 🗣️ Answering Judge Questions

**"Why ensemble GNN + XGBoost?"**
GNN captures non-linear structural patterns (ring topology, bond angles) that molecular fingerprints miss. XGBoost captures global statistical correlations in ECFP space. They're complementary — their errors don't overlap, so averaging them reduces variance and improves AUC.

**"What does high logP mean for toxicity?"**
High lipophilicity means the molecule accumulates in lipid membranes and fatty tissues (liver, brain). This extends exposure time and increases the chance of off-target binding and organ damage. logP > 5 is a Lipinski violation and a toxicity risk factor.

**"What are toxicophores?"**
Structural fragments known to cause reactivity or metabolic activation: nitro groups (form reactive nitroso intermediates), epoxides (alkylate DNA), aromatic amines (activate to arylnitrenium ions), quinones (redox cycling). Detecting them is the first step before modelling.

**"Why multi-task learning on 12 endpoints?"**
Toxicity mechanisms overlap. A molecule toxic to NR-AR often shares structural features with SR-MMP-toxic molecules. Multi-task learning forces the model to learn shared representations, improving performance on endpoints with fewer labelled examples (positive transfer).
